## What is Space Invaded?
- Space Invaded is the newest version of Space games with better features.\
Including the classic Space Invaded.\
This game is an adventurous Space Invaded edition.\
We made the game greater and better than before.\
This is a revolution for the Arcade Gaming market.


</br>

---

</br>

## Features:
- The classic Space Invaded
- Space Invaded (Modern)
- Retry button
- Options button
- Startup screen
- Info about the controls
- The greatest Space game on the Arcade Gaming market.
- High Scores
